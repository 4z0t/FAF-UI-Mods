# Rings For All

A successor to mods like Ranges and any other mod that show Ranges of units (including Smart ring display from me). It uses new engine functionality to draw rings in world view, which are far more clearer compared to predecessors.

![rfa1](/Media/rfa1.png)

You can see attack radius of enemy buildings, for example SMD

![rfa2](/Media/rfa2.png)

Mod assists during reclaiming

![reclaim](/Media/RFA.gif)

In options menu you can select which types of ranges display and which preview keys to use.
