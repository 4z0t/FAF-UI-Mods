# ReUI.ActionsPanel

A configurable panel is integrated into the bottom-right UI. This panel serves as a centralized hub for actions defined by installed extensions, allowing for quick execution.

![ActionsPanel](/Media/enhacements.png)

## Settings

Panel has various settings for adjusting its size.

![settings](/Media/grid.png)

## Extensions

Panel is extension based. You can make your own, or use ones that are already created. You can select ones you want to use in options.

![extensions](/Media/extensions.png)

## ACU Enhancements

Shows available enhancements for your ACU and SACU with no need to navigate through construction panel tabs.

![enhancements](/Media/enhacements.png)

## Factory templates

Shows available factory templates for currently selected factories.

![templates](/Media/templates.png)
